Bulk import

Collections

This should be a CSV file with the following column headings: "name", "description", "count", "cabinet", "drawer". The headings are case insensitive. Additional columns are ignored. The "cabinet" and "drawer" columns specify the cabinet name and drawer label respectively. This is the same format as the collections export.

Resource Links

This should be a CSV file with the following column headings: "name", "collection" and "url". The headings are case insensitive. Additional columns are ignored. The "collection" columns specifies the collection name. This is the same format as the links export.
